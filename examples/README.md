# OpenAGI Examples

This directory contains example scripts demonstrating how to use OpenAGI features.

## Available Examples

### Basic Usage
- `basic_usage.py` - Demonstrates platform initialization and basic feature execution

### NLP Examples
- `nlp_pipeline.py` - Complete text processing pipeline
- `sentiment_analysis.py` - Advanced sentiment analysis examples

### Computer Vision Examples
- `image_processing.py` - Basic image processing workflows
- `object_detection.py` - Object detection examples

### Machine Learning Examples
- `classification.py` - Text and data classification examples
- `clustering.py` - Data clustering examples

## Running Examples

```bash
# Run basic usage example
python examples/basic_usage.py

# Or using the CLI
openagi demo
```

## Requirements

All examples require the OpenAGI package to be installed:

```bash
pip install -e .
```
