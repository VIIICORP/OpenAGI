### 🛠 Available Commands

```bash
# Platform Management
openagi version                    # Display version info
openagi config                     # Show configuration
openagi server start               # Start OpenAGI server

# Testing Framework  
openagi test run --suite comprehensive  # Run all 30M+ tests
openagi test run --suite quick         # Run quick validation
openagi test history                   # View test history
openagi test report                    # Generate test reports

# Model Management
openagi models list                # List available models
openagi models download <model>    # Download AI models
openagi models load <model>        # Load models into memory

# Database Operations
openagi db init                    # Initialize database
openagi db migrate                 # Run migrations
```
